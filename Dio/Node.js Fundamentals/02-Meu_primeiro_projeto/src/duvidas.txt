comandos terminal

navegação 
------------------------------
del * - deleta arquivos de um apasta (todos! cuidado!)
ls - lista os arquvios de um diretório
cls - clear (limpa o terminal)
cd - muda o diretório
mkdir - cria diretorio
code - abrir o vscode na pasta
code. - abre o vscode

npm e node
------------------------
npm init -y inicia um projeto node sem as perguntas
node[arquivo] executa o arquivo js expecificado
node --watch monitora o arquivo expecificado (vigia) crtl +c desliga ele
